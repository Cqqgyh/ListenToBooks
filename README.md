# 硅谷听书

这是一个音频分享平台，用户可以创建自己的音频专辑，分享给其他用户，也可以订阅其他用户的专辑，听取他们分享的音频。

技术体系：uni-app、vue3、pinia、GraceUI、微信小程序、腾讯云、新浪云等
